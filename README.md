# TestNG Framework Examples

This repository contains practical examples of using **TestNG annotations** to manage test execution flow. It is designed for QA engineers and developers learning how to structure and control automated tests.

## 📌 Features Demonstrated
- `@Test` annotation basics
- `dependsOnMethods` for method-level dependencies
- `dependsOnGroups` for group-level dependencies
- `enabled` to include/exclude tests
- `alwaysRun` to force execution despite dependencies
- Grouping tests with `@Test(groups = "...")` and `testng.xml`
